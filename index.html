INTÉGRATION WEB

Date de création :

2019


Contexte :

Projet d’intégration réalisé pendant mon alternance dans l’agence Avent Media Groupe.


Client :

Cafe Royal


Objectif du site :

Envoyer les utilisateurs vers le site client pour augmenter le trafic grâce aux codes promos. Après s'être inscrit l'utilisateur reçoit une invitation par mail afin d'accéder au site de confirmation.

La campagne a été déclinée en 3 langues : française, Anglaise et Espagnol.


Technologies utilisées :

HTML - CSS - SASS - Responsive - PHP - Gitbash - Avocode
